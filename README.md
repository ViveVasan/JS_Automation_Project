# JS_Automation_Project

### Environment Setup

1. Global Dependencies
    * Install [Node.js](https://nodejs.org/en/)
    * Or Install Node.js with [Homebrew](http://brew.sh/)
    ```
    $ brew install node
    ```

2. Project Dependencies
	* Install Node modules
	```
	$ npm i init
	```

### Running Tests
